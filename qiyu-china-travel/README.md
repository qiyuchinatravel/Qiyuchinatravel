# QIYU China Travel - 奢华定制旅行网站

这是一个功能完整的品牌形象与行程定制平台，专为高端旅行者设计。

## ✨ 主要功能

- **奢华品牌展示**：响应式首页，展示精选行程、目的地、客户评价。
- **智能行程定制器**：拖拽式构建个性化行程，实时计算费用。
- **多语言支持**：英语、法语、德语、西班牙语、中文。
- **完整后台管理**：
  - 内容管理（城市、景点、酒店、餐厅、交通、导游）
  - 行程管理（创建/编辑/删除旅游套餐）
  - 博客系统（多语言富文本编辑、标签、SEO）
  - 预订/咨询管理（状态跟踪、备注）
- **富文本编辑器**：所有描述字段均支持格式化排版。
- **资源快捷添加**：在行程创建弹窗中可快速添加城市/景点/交通等资源。

## 🛠️ 技术栈

- 纯原生 HTML5 / CSS3 / JavaScript（无外部依赖）
- 使用 Google Fonts 和 Unsplash 图片占位
- 数据存储于前端内存（演示用，可对接后端API）

## 🚀 快速开始

1. 克隆或下载本仓库。
2. 直接用浏览器打开 `index.html` 即可运行。
3. 后台管理登录：点击右上角 `Admin`，任意用户名/密码（例如 `admin` / `admin123`）。

## 📁 文件说明

- `index.html`：单文件应用，包含所有样式、脚本和结构。
- 无需额外图片或资源，所有图片均来自 CDN 或 Unsplash。

## 🔧 自定义开发

- 数据模型位于 `DATA` 对象中（行程、目的地、城市、景点等）。
- 博客内容存储在 `blogPosts` 数组。
- 行程定制器的状态保存在 `state.itinerary`。
- 所有富文本编辑器通过 `initRTE()` 函数动态生成。

## 📄 许可

仅供学习和演示使用。若用于商业目的，请替换图片、品牌标识并确保数据安全。

## 🌐 在线预览

由于是静态HTML，可直接部署到任何 Web 服务器或 GitHub Pages。
